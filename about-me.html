About Me

Hello there! I'm currently a student following the software developer traject and I have made this game as a project on my school. When I had to think of making a website my own, the idea popped into my head to make a dabloon game once, highly inspired by the dabloon memes.

This is an RNG-based game where the events base off of an RNG, where you, the player, can use items and dabloons to advance further into the game, while also still being able to use these items to keep control over the game. Its not very large, but you can still achieve a lot. I might start adding more features if this will get more popular
